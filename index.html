Key Financial Formula

Return on Equity (ROE):

$$ \mathrm{ROE} = \frac{\mathrm{Net\ Income}}{\mathrm{Shareholders'\ Equity}} = \frac{NI}{E} $$

DuPont decomposition: $$ \mathrm{ROE} = \mathrm{Profit\ Margin} \times \mathrm{Asset\ Turnover} \times \mathrm{Equity\ Multiplier} $$ $$ \mathrm{ROE} = \left(\frac{NI}{Revenue}\right)\left(\frac{Revenue}{Assets}\right)\left(\frac{Assets}{Equity}\right) $$

Explain DuPont: how margin, turnover and leverage combine to drive ROE.

Example: Calculating CAGR (Python)


# CAGR calculation
def cagr(begin_value, end_value, periods):
    return (end_value / begin_value) ** (1.0 / periods) - 1

# Example
start = 100
end = 150
years = 3
print(f"CAGR: {cagr(start, end, years):.4%}")
