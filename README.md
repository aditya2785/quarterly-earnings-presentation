# Quarterly Earnings — Reveal.js Presentation

This repository contains a Reveal.js presentation for the quarterly earnings report.

## How to preview locally
1. Install a static server (e.g., `npm i -g http-server`) or use Python:
   - Python 3: `python -m http.server 8000`
2. Open `http://localhost:8000/` in your browser.

## How to publish via GitHub Pages
1. Create a repo on GitHub named `quarterly-earnings-presentation`.
2. Commit and push these files to the `main` branch.
3. In the repository Settings → Pages:
   - Source: `main` branch
   - Folder: `/ (root)` or `docs/` (if you put files in `docs/`)
4. Wait a minute and visit:
   `https://<YOUR-GITHUB-USERNAME>.github.io/quarterly-earnings-presentation/`

Add `?v=1` to bypass caches.
